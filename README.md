****CourseZone - Next.js Dynamic Routing Project****

A sleek, mini-project built to master Dynamic Routing in Next.js. This project focuses on navigating through different course modules seamlessly without a complex backend.

=> Key Features

Dynamic Routes:

Uses Next.js [slug] or [id] routing to generate course-specific pages on the fly.
Modern UI: Features a clean interface with Linear Gradient backgrounds and interactive hover-effect cards.

Detailed Information:

Clicking a card (e.g., Python) displays specific course data like duration and learning objectives.

=> Tech Stack

Framework: Next.js (App Router)
Styling: Tailwind CSS / CSS Gradients
Logic: JavaScript Object Mapping (Mock Data)
